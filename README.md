
# Como executar o projeto

```bash
# clonar repositório
git clone https://github.com/progmateus/hiper-challenge.git

# entrar na pasta do projeto back end
cd hiper-challenge

# instalar dependências
yarn install

# executar o projeto
yarn start
```
